# Facial-Emotion-Detection-System
This project focuses on emotion detection using computer vision to classify facial expressions such as neutral, angry, disgust, fear, happy, sad, and surprise. It enables real-time analysis of human emotions, providing insights for applications in psychology, security, and human-computer interaction.
